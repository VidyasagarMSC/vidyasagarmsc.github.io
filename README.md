### Personal website of Vidyasagar (Sarath Chandra) Machupalli

Built with modern design patterns using **Vite**, **vanilla JavaScript**, and **CSS custom properties**. Features dark/light mode, responsive design, interactive research portal with knowledge graph visualization.

**Tech Stack:** Vite · Vanilla JS · CSS Custom Properties · Font Awesome · Google Fonts

**Development:**
```
npm install
npm run dev
npm run build
```
